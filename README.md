# BudgetVault
BudgetVault is a modern expense tracking and budget management web application designed to help students and individuals manage their daily spending efficiently. The application provides a clean dashboard interface for recording expenses, monitoring budgets, and understanding spending patterns in a simple and organized way.

## Live Demo
https://budget-vault-ochre.vercel.app/

## Features
* Add and manage daily expenses
* Set monthly budget limits
* Track spending categories
* Real-time expense updates
* Firebase database integration
* Responsive dashboard design
* Modern and clean UI
* Expense summary and analysis

## Tech Stack
* HTML5
* CSS3
* JavaScript
* Firebase - Firestore
